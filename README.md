# Wav Mastering Studio

WAVファイルをブラウザの中だけで簡易マスタリングするローカルWebアプリです。

## できること

- WAVファイルを読み込む
- Balanced / Warm / Punchy / Loud のプリセットを選ぶ
- 明るさ、低音の太さ、まとまり感、書き出し上限を調整する
- 元音とマスタリング後を聴き比べる
- マスタリング後のWAVを書き出す

## 使い方

このフォルダで次のコマンドを実行します。

```bash
python3 -m http.server 4173
```

ブラウザで次を開きます。

```text
http://localhost:4173
```

## 大事なこと

音の処理はすべて自分のパソコン内で行います。読み込んだWAVファイルは外部サーバーに送られません。

これはプロのマスタリングエンジニアの作業を完全に置き換えるものではなく、デモ音源やSNS投稿前の音量・質感調整を手早く行うためのツールです。
